# Nmap Scripts for a Protocol
Bash script to show the scripts that nmap has for a specific protocol. 

# Installing
```bash
git clone https://github.com/h3g0c1v/nps.git
cd nps
chmod +x nps.sh
./nps.sh
```

# Use

Its use is as follows

```bash
.\nps PROTOCOL1 [PROTOCOL2 PROTOCOL3 ...]
```
